Autocite - A Batch Citation Macine

What is AutoCite?

AutoCite is a website citation machine that accepts URLs to websites and returns citations to the websites in either Chicago or APA format. It was created by Brandon Tang to ease the process of creating citations for papers. With AutoCite, you can focus more on the paper and less on the references.

Why AutoCite?
  • Fast Citations
  • Legit Looking Citations (most of the time)
  • Straightforward to Use
  • FREE (At least until my AWS credits run out)
    • Limitations of AutoCite
  • AutoCite is unable to cite other types of sources (E.g. PDF Documents, Journals, etc)
  • Some websites do not work with AutoCite
  • Autocite can only find a max of 1 author and may not even find authors in many cases
    • How does AutoCite Work?
    - Autocite runs most of its code on AWS lambda.
    - URLs to be cited are submitted 1 at a time to an API Gateway that triggers AWS lambda.
    - Lambda function then uses python to request the source code from the website and uses regular expressions to extract the information required.
    - The cited URLs are stored as a cache in a MySQL server on AWS RDS for future reference.
    - This cache is checked everytime the "generate citations" button is clicked and updated once all the citations have been made.
